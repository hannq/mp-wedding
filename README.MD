# 婚礼小程序

## 项目概览

```
.
├── README.MD
├── client                  客户端
├── cloud                   小程序云开发
├── project.config.json     小程序配置文件
└── scripts
```

## 安装

### 客户端

```bash
cd ./client
pnpm i
```

### 云开发

```bash
cd ./cloud
pnpm i
```

## 开发

### 客户端

```bash
cd ./cloud
npm run dev:weapp
```

### 云开发

```bash
cd ./cloud
npm run dev
```


## 构建

### 客户端

```bash
cd ./cloud
npm run build:weapp
```

### 云开发

```bash
cd ./cloud
npm run build
```
